# Multiplatform App Development Exercises

This repository contains a collection of exercises and mini-projects from my Multiplatform App Development course. These exercises showcase my skills in various areas, including:

- **Programming Languages:** Java, JavaScript
- **Technologies:** Web and mobile app development, databases
- **Cloud Concepts:** Introduction to AWS (optional)

Each folder within this repository corresponds to a specific exercise or mini-project, with detailed explanations and the technologies used.

## Getting Started

To get started, clone this repository to your local machine and navigate through the folders to explore each exercise.
